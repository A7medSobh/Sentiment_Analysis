# Sentiment_Analysis

## 1. Execution Instructions

### Requirements

- ##Node.js## must install it on your computer

### File Setup

Make sure the following files are in the same directory as the script ('main.js'):

1- 'socialsent.csv' ( the dictionary of word scores)
2- '.txt' review files you wish to test

### HOW TO RUN?:

Open your terminal and run the following :

bash:

	node main.js <file_name>

	(Note: If no <file_name> is provided, the program will default to searching for a file named review.txt)
